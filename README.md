# 🎬 Netflix Clone (Frontend Project)

## 📌 Overview
This project is a simple Netflix Clone developed using **HTML, CSS, and JavaScript**. It is a single-page frontend application that replicates the UI of Netflix, including homepage layout, banners, and movie sections.

This project was created for learning purposes to improve frontend development skills and UI design understanding.

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## ✨ Features

✔ Responsive Netflix-like UI  
✔ Hero banner section  
✔ Movie thumbnails layout  
✔ Hover effects on cards  
✔ Clean and modern design  
✔ Single-page frontend application
